Kurzbeschreibung

Die CSS-Target-Technik ermöglicht die Erstellung eines Overlay-Menüs für mobile Webseiten ohne JavaScript. Hierbei wird das :target Pseudo-Selektor verwendet, um den Zustand des Menüs basierend auf der URL-Ankerreferenz zu steuern. Durch Klicken auf einen Link mit der entsprechenden Ankerreferenz wird das Menü angezeigt oder ausgeblendet.
